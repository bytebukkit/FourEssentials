# FourEssentials
FourEssentials aims at bringing features from the [Essentials](https://essentialsx.net) plugin of Java Edition and add it to FourKit.

## Commands

- `heal` — restore your health  
- `feed` — refill your hunger  
- `fly` — toggle flight  
- `god` — enable invulnerability  
- `suicide` — end your own life  
- `more` — fill the held stack to max  
- `repair` — fix durability on one or all items  
- `speed` — change your movement speed  
- `getpos` — show your coordinates or another player’s  
  - aliases: `pos`, `coords`, `position`  
- `exp` — give set reset or view experience  
  - alias: `xp`  
- `gc` — show memory uptime and tick info  
- `ping` — check your latency  

## Gamemode

- `gamemode` — change a player’s gamemode  
  - alias: `gm`  
- `gms` — set survival mode  
- `gmc` — set creative mode  
- `gma` — set adventure mode  

## Teleportation

- `tp` — teleport to a player  
- `tphere` — bring a player to you  
  - alias: `tph`  
- `tppos` — teleport to coordinates  
- `tpall` — teleport all players to someone  
- `back` — return to your previous location  
- `spawn` — teleport to world spawn  

## Homes

- `sethome` — set your home  
- `home` — teleport to your home  
- `delhome` — remove a home  
- `homes` — list your homes  

## Warps

- `setwarp` — create a warp  
- `warp` — list or teleport to a warp  
- `delwarp` — delete a warp  
- `warps` — list all warps  

## Chat and Identity

- `nick` — change your nickname  
- `msg` — send a private message  
  - aliases: `tell`, `w`, `whisper`, `pm`  
- `r` — reply to the last message  
  - alias: `reply`  
- `me` — send an action message  
- `broadcast` — send a server‑wide message  
  - aliases: `bcast`, `bc`  
- `afk` — mark yourself as AFK  
- `list` — show online players  
  - aliases: `who`, `online`  
- `seen` — show when a player was last online  
- `whois` — show basic info about a player  
- `realname` — find a username from a nickname  
- `vanish` — hide yourself from others  
  - alias: `v`  
- `ptime` — change your personal time  

## Moderation

- `kick` — remove a player  
- `kickall` — remove all players  
- `ban` — ban a player  
- `mute` — mute or unmute a player  
- `unmute` — unmute a player  
- `kill` — kill a player

Please respect the fact that not everything could be tested properly, and this is accessible to all players on the servers.
Feel free to PR fixes and new features.
